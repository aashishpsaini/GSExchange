# README

This application is simple prototype where it represents an simple exchange.
One Assumption here is: 
 - Sellers data is already present.
 - Buyers when come for buying they request gold and payload looks like:

  {
    "buyer": {
      "buyer_name":"Buyer-1",
      "time_stamp":"9:01",
      "rate":1200,
      "amount":30
    }    
  }
